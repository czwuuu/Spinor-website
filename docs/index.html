旋量简介

旋量（Spinor）是物理学和数学中的重要概念，最初由数学家和物理学家埃利·嘉当（Élie Cartan）在研究李群的表示时引入。在量子力学中，旋量用于描述具有半整数自旋的粒子（如电子、质子等）的量子态。

旋量在相空间旋转360°后会变为原来的负值，需要旋转720°才能回到原始状态，这一特性使其区别于普通矢量。
